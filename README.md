# 芒果博客
基于vue技术栈开发的个人博客项目，用于巩固所学知识。暂时停止该项目开发

使用markdown语法来编写文档，通过插件实现markdown富文本编辑和实时预览。

# 项目技术栈
- vue-cli 4.2.2：采用ui界面创建项目及安装依赖
- vue
- vue-router
- vuex --- 暂时使用不到
- element-ui
- axios
- mavon-editor：markdown富文本编辑器插件
- less
- eslint
